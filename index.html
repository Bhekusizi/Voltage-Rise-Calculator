Copper conductors: Tables 4.5(A) / 4.7(A) · Aluminium conductors: Tables 4.5(B) / 4.7(B)
PVC insulation → 75°C operating temperature (Col 5)
XLPE insulation → 90°C operating temperature (Col 7 / Col 6)
X-HF-110 / EPR insulation → 110°C operating temperature (Col 8 / Col 7)
110°C cables use XLPE reactance values from Table 4.1 (cross-linked insulation geometry)
Single-core reactance: flat touching formation (Table 4.1(A)) — worst case for wiring enclosures
Multicore reactance: circular conductors (Table 4.1(B))
Voltage drop uses Vc = Rc·cosφ + Xc·sinφ method (AS/NZS 3008.1.1 Appendix C) — accounts for power factor angle
Current from kW: I = P/V (1φ) or P/(√3V) (3φ) — PF not used in current derivation from real power

This calculator determines voltage rise only — it does not verify current-carrying capacity (CCC). A compliant voltage rise result does not confirm the cable is adequately sized. CCC must be independently verified per AS/NZS 3008.1.1:2025 (Tables 3.5–3.31) with all applicable correction factors per AS/NZS 3000:2018 Clause 3.5.

Effective voltage drop coefficient (Ω/km):

Vc = Rc × cosφ + Xc × sinφ

Single-phase voltage rise (V):

ΔV₁φ = I × L × 2 × Vc / 1000

Three-phase voltage rise (V):

ΔV₃φ = I × L × √3 × Vc / 1000

Percentage voltage rise:

%ΔV = (ΔV / V_nominal) × 100

Cable impedance magnitude (Ω/km) — shown for reference only:

Zc = √(Rc² + Xc²)

Current from real power (kW):

I₁φ = P / V · I₃φ = P / (√3 × V)

Current from apparent power (kVA):

I₁φ = S / V · I₃φ = S / (√3 × V)

Where I = current (A), L = cable length (m), Vc = effective voltage drop coefficient (Ω/km), Rc = AC resistance, Xc = reactance, φ = power factor angle. The Vc method (R·cosφ + X·sinφ) accounts for the phase angle between current and voltage, giving a more accurate result than using impedance magnitude Zc alone. Resistance values from AS/NZS 3008.1.1:2025 Tables 4.5(A) & 4.7(A). Reactance from Tables 4.1(A) & 4.1(B). Voltage rise limit per AS/NZS 4777.1:2024 Cl. 4.5.
